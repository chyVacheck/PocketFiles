# Security Policy

## Supported Versions

PocketFiles is currently under active development and has not yet reached its first stable release.

Security fixes are applied to the latest development version only.

## Reporting a Vulnerability

If you believe you have found a security vulnerability in PocketFiles, please do not open a public GitHub issue.

Instead, report it privately by email:

**dmytro.4or.dev@gmail.com**

Please include:

- A description of the vulnerability.
- Steps to reproduce it.
- The affected PocketFiles version.
- Any relevant logs or proof of concept.

We will investigate the report and work on an appropriate fix before public disclosure.

Thank you for helping keep PocketFiles secure.
