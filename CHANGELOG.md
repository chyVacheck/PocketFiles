# Changelog

All notable changes to PocketFiles will be documented in this file.

## [0.1.0] - 2026-07-24

### Added

- Local filesystem storage.
- SQLite metadata persistence.
- Physical files and file usages.
- SHA-256 file deduplication.
- Configurable date-based directory layout.
- File save and open operations.
- Soft delete and restore operations.
- Permanent file purge.
- Physical file lifecycle management.
- Public `PocketFiles` facade.
- Automated tests and GitHub Actions build.
